# 智能闪卡学习系统

一个基于遗忘曲线的闪卡复习工具，主要是为了自己备考用的。

## 为什么做这个

之前用过Anki之类的工具，功能是挺强的，但是感觉太重了，而且界面不太合我口味。所以就想自己写一个轻量级的，能在手机浏览器里直接用的那种。

核心思路就是利用间隔重复的原理，让你在快要忘记的时候复习，这样记忆效果最好。算法参考了SM-2，不过我根据自己的使用体验做了一些调整。

## 主要功能

### 间隔重复

每张卡片会根据你的评分（1-5分）自动安排下次复习时间。评分低的话很快会再出现，评分高的话间隔会越来越长。

具体间隔大概是这样：

- 1-2分：重新开始，1天后再见
- 3分：保持当前间隔
- 4-5分：间隔翻倍（1天→2天→4天→8天→16天→32天...）

### 富文本

题目和答案支持Markdown格式，包括：

- 代码块（有语法高亮）
- 图片（通过URL引用）
- 表格
- 数学公式（简单的）
- 粗体、斜体、删除线等

### 标签系统

可以给卡片打标签，然后按标签筛选。比如你可以把所有"数据结构"的卡片打上标签，单独复习这一块。

### 搜索

支持全文搜索，题目、答案、标签、章节都能搜。有搜索历史记录，方便重复搜索。

### 离线使用

这是一个PWA应用，加到手机主屏幕后可以离线使用。数据存在浏览器本地，不会上传到任何服务器。

## 文件结构

```text
enhanced-flashcard-pwa/
├── index.html          # 入口页面
├── app.js              # 主逻辑（约500行）
├── spaced-repetition.js # 间隔重复算法
├── richtext.js         # Markdown渲染
├── tags.js             # 标签管理
├── search.js           # 搜索功能
├── styles.css          # 样式
├── manifest.webmanifest # PWA配置
├── sw.js               # Service Worker
├── data/
│   └── cards.js        # 题库数据（需要自己填充）
└── assets/
    └── icon.svg        # 应用图标
```

## 怎么用

1. 直接打开 `index.html` 就行
2. 手机上建议"添加到主屏幕"，体验更好
3. 题库在 `data/cards.js` 里，按格式添加自己的题目

## 题库格式

```javascript
{
  id: "唯一ID",
  question: "问题内容（支持Markdown）",
  answer: "答案内容（支持Markdown）",
  mode: "subjective",  // subjective=主观题, objective=客观题
  tags: ["标签1", "标签2"],
  chapter: "章节名",
  number: "题号"
}
```
